# Character Length Checker

Character Length Checker is a simple yet practical web application built using **HTML, CSS, and JavaScript**. This tool allows users to enter any text and instantly view the total number of characters in real time. Unlike many basic counters, this application also counts **spaces, punctuation marks, numbers, and special characters**, ensuring an accurate representation of the complete text length.

The project features a clean and responsive user interface, making it easy to use on both desktop and mobile devices. As users type, the character count updates dynamically without requiring a page refresh, providing a smooth and interactive experience. The application demonstrates core front-end development concepts such as **DOM manipulation, event handling, user input processing, and real-time updates using JavaScript**.

## ✨ Features

- Real-time character counting
- Counts spaces along with all other characters
- Responsive and user-friendly design
- Instant updates while typing
- Clean and modern interface
- Built with pure HTML, CSS, and JavaScript
- Beginner-friendly project for learning front-end development concepts
- Accurate counting of letters, numbers, symbols, punctuation, and spaces

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🎯 Purpose

This project was created to practice front-end web development skills and demonstrate how JavaScript can be used to create interactive web applications. It can be useful for checking text length for social media posts, form inputs, content writing, assignments, and any situation where character limits matter.

By building this project, developers can gain hands-on experience with:
- DOM manipulation
- Event listeners
- User input handling
- Dynamic content updates
- Responsive web design
